# 更新日志

本项目所有重要变更记录于此文件。版本号遵循语义化版本规范。

## [1.1.0] - 2026-06-08

### 新增

- **详情页三色堆叠柱状图**：Flash / Pro 详情页"按日 Token 消耗"改为缓存命中（青色）/ 缓存未命中（橙色）/ 输出 Token（紫色）三段堆叠柱，各段高度按实际 token 比例计算，保留最小可见高度。
- **详情页 hover 浮窗**：鼠标悬停柱子时显示磨玻璃浮窗，展示当日缓存命中量、缓存未命中量、输出 token 及命中率百分比。
- **模型卡片缓存命中率**：主界面 Flash / Pro 卡片下方新增当月缓存命中率百分比（输入 token 中命中缓存的占比）。
- **主界面缓存命中明细图表**：主面板底部新增"缓存命中明细"柱状图，将 Flash 与 Pro 合并展示，hover 浮窗风格与整体 UI 一致。
- **亮色玻璃皮肤**：新增一套亮色半透明磨玻璃主题，面板、卡片、图表、设置页配色全面适配。
- **UI 外观切换器**：主界面顶部 Shirt（衣服）图标改为一键切换，默认暗色，点击切到亮色，再次点击恢复暗色；选择结果通过 `localStorage` 持久化，重启后自动恢复且无首帧闪烁。
- **设置页版本信息**：设置页新增"关于 / 当前版本"，直接读取应用版本号并显示。

## [1.0.1] - 2026-06-07

### 修复

- 修复应用单实例缺失导致的重复多开问题。此前在程序已运行的情况下，再次点击图标或 `exe` 会不断启动新的进程，且可以无限叠加（主窗口 `skipTaskbar` 不进任务栏，多开后不易察觉）。现接入 `tauri-plugin-single-instance` 单实例守卫：再次启动时不再新开窗口，而是把已有主面板显示并聚焦，第二个进程随即自行退出。

## [1.0.0] - 2026-06-07

### 新增

- 首个正式发布版本。
- DeepSeek API 账户余额查询，使用 DeepSeek 官方余额接口。
- DeepSeek 平台用量查询：当月消费、模型 Token 总量、请求数、缓存命中、缓存未命中、输出 Token。
- 支持 V4 Flash 与 V4 Pro 两类模型用量展示。
- 最近 7 天消费趋势图与模型详情页。
- Windows 托盘入口，主窗口默认不进入任务栏。
- API Key 保存、清除与余额验证。
- 用量 Token 网页登录自动同步与手动粘贴兜底。
- NSIS 安装包，安装程序与主程序均使用鲸鱼 logo 图标。
